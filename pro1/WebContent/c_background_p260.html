색상(p256)과 배경(p260)

  • background-color: 배경색
    0 a valid color name - like "red"
    a HEX value - like "#ff0000"
    an RGB value - like "rgb(255,0,0)"
    opacity 속성은 투명도 0.0(투명)~1.0(불투명)
  • background-image:url(경로 및 파일명)
  • background-repeat: 이미지 반복
    background-repeat: repeat(defalut)|repeat-x|repeat-y|no-repeat
  • background-attachment: 배경이미지고정여부
    background-attachment: scroll|fixed|local|initial|inherit;
  • background-position: 배경이미지 위치를 의미
