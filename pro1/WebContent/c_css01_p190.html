스타일 우선 순위

1. 사용자 스타일 시트가 최우선
2. !important가 붙은 스타일
3. 사용자 정의 일반스타일(제일 빈번)
4. 기본적인 브라우저 스타일 시트

CSS의 선언방식 3가지

*internal방식:html문서안에 head요소, body요소에 에 style선언 문법- <head> <style> 선택자 {css:속성명:값;css:속성명:값;css:속성명:값;} </style> </head> *external방식:html 문서 밖에서 별도의 css를 선언 문법-<head> <link rel = "stylesheet" href = "경로\파일명.css"> </head> *inline 방식: 특정 요소에 style 속성을 이용 문법 - <태그명 style="css속성명:값"></태그명> </li>
