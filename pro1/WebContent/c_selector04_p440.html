연결선택자

  • 구조가상클래스(p440)
    구조가상클래스란? 웹문서 구조를 기준으로
    특정 위치에 있는 요소를 찾아 스타일을 지정할때 사용하는
    가상 클래스 선택자이다.
  • 요소명:nth-child(odd) : 홀수번째 요소를 의미
  • 요소명:nth-child(even) : 짝수번째 요소를 의미
  • 요소명:nth-child(n) : n번째 요소를 의미
    요소명:nth-child(1) : 1번째 요소
  • 요소명:nth-child(n+1) : n+1번째 요소를 의미
    요소명:nth-child(1) : 1번째 요소
  • 요소명:nth-child(2n+3) : n+1번째 요소를 의미
    요소명:nth-child(1) : 1번째 요소
  • 요소명:nth-child(-n+3) : n+1번째 요소를 의미
    요소명:nth-child(1) : 1번째 요소
  • 요소명:nth-child(1+3):nth-child(-n+9) : n+1번째 요소를 의미
    요소명:nth-child(1) : 1번째 요소
  • 요소명:nth-child(3n) : 3n번째 요소를 의미
    3*n으로 의미 3*1 = 3 3*0 = 0 요소명:nth-child(1) : 1번째 요소
  • 의사클래스(가상클래스)
    의사클래스는 선택자에 추가되는 키워드
    선택한요소가 특별한 상태여야 만족할수 있다 예):hover
  • 자식선택자 => 부모요소 > 자식요소
    지정한 자식요소에 스타일 적용
  • 인접형제 선택자(p425) => 동등한형제요소+동등한형제요소
    형제중에서 가장 가까운 형제 요소에 스타일 적용
  • 형제 선택자(p427) => 요소~요소
    기준요소 뒤의 모든 형제요소에 스타일 적용
